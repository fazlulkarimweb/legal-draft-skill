# Trademark Description Drafting Skill
## 1. Trigger Context
Activate when preparing trademark applications or descriptions of marks.
## 2. Formatting & Structure
- **Nice Classification:** Identify the correct Nice Class (e.g., Class 9 for software, Class 42 for SaaS).
- **Mark Description:** Describe the mark, including any design elements, colors, or stylized text.
- **Identification of Goods/Services:** Write clear, precise descriptions of the goods or services associated with the mark.
## 3. Operational Rules
- Cross-reference terms with the USPTO ID Manual or equivalent national registries to prevent rejections.
- Avoid overly broad terms; define specific, clear classes of goods or services.
